Acerca de la materia

Este curso, impartido por el maestro Guillermo Calderón, es el primero en la secuencia de desarrollo web y cubre los fundamentos básicos de los lenguajes HTML, CSS y JavaScript, que son los elementos esenciales de cualquier aplicación web. Las unidades de esta materia cubren los siguientes contenidos: introducción al HTML y al CSS, maquetación de sitios web, frameworks de diseño web responsivo, introducción a JavaScript, el modelo de objetos de documento y JavaScript asíncrono.
